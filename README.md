# ElGamal---kriptografija
Projekat iz kursa "Kriptografija" - enkripcija i dekripcija ElGamal algoritmom.

## Grupe zadataka

**1. Grupa I**
- Web Aplikacija
    * User-friendly interface
    * Rad sa fajlovima (upload fajla, download fajla)
    * Pokretanje Python funkcija iz UI-a
    * Prikaz rezultata

- Dokumentacija (kako se koristi aplikacija)


**2. Grupa II**
- Alisin dio
    * Python skripta za *prime-num_find* funkciju (implementacija miller_rabin)
    * Python skripta za *elgam_set* funkciju
    
- Dokumentacija
    * Opisati postupak implementacije i rad funkcija
    * Navesti ograničenja ili nedostaci programa, ako postoje
    * Navesti poboljšanja postojećeg programa, ako postoje


**3. Grupa III**
- Bobov dio
    * Python skripta za *elgam_enc* funkciju
    * Python skripta za *elgam_dec* funkciju

- Dokumentacija
    * Opisati postupak implementacije i rad funkcija
    * Navesti ograničenja ili nedostaci programa, ako postoje
    * Navesti poboljšanja postojećeg programa, ako postoje

## Alati

- Web aplikacija
    * HTML/CSS
    * JS za logiku i slanje zahtjeva backend-u

- Backend
    * FastAPI
        - ideja je da funkcije budu API
        - u JS će se samo slati zahtjevi na definisane rute
